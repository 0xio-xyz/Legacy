# Changelog

All notable changes to the 0xio Wallet (Legacy) are documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adhered to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

**Note:** This project is archived. No further updates will be made.

## [1.2.2] - 2025-11-24

### Changed
- Enhanced DApp connection management
- Improved transaction retry logic
- Better error handling and diagnostics

### Fixed
- UI/UX improvements
- Bug fixes and performance optimizations

## [1.2.1] - Previous Release

### Added
- Multi-wallet support with easy wallet switching
- Private balance encryption and decryption
- Private transfers with end-to-end encryption
- DApp connectivity via SDK integration
- Transaction history tracking
- Custom network support
- Secure password protection
- Import/Export wallet functionality
- QR code support for easy address sharing

### Features
- Manifest V3 Chrome extension
- Non-custodial wallet architecture
- BIP39 mnemonic phrase support
- Session-based unlocking (30-minute timeout)
- Bulk transfer capabilities
- Transaction retry mechanism

### Security
- SHA-256 password hashing
- Encrypted private key storage
- Chrome's secure storage API
- No external data transmission

## [1.0.0] - Initial Release

### Added
- Initial wallet implementation
- Basic send/receive functionality
- Octra Network integration
- Browser extension framework
